iTV: Fetch TV shows on Internet
===============================

This tool downloads TV shows on Internet.

QuickCheck Specification
------------------------

iTV includes a set of QuickCheck properties which act as both
tests and a formal specification (hence their inclusion in the Haddock
documentation).  To run the tests do:

   cabal configure --enable-tests
   cabal build
   cabal test
